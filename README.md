# korea-monthly-report-sp

Hosting for the monthly **Korea Sales & Stock Report** sent to the supplier.

**This repository contains ciphertext only.** Each report page is encrypted with
AES-256-GCM (key derived via PBKDF2-SHA256, 600,000 iterations) and can only be
read by entering the password in the browser. No plaintext sales data is stored here.

Pages are generated by `publish.py` in the private workspace repo — do not edit
`<YYYY-MM>/index.html` by hand.
